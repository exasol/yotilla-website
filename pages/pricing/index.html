Metering

The unit in which consumption is measured is the total number of deployed Data Mart attributes per month. The number of deployed Data Mart attributes for an Yotilla instance per month is the maximum of deployed Data Mart attributes over all environments (e.g., development and production) of the Yotilla instance per month. The total number of deployed Data Mart attributes is the sum of deployed Data Mart attributes over all instances per month.

Each package includes a fixed number of deployed Data Mart attributes. Metering of the additional deployed Data Mart attributes takes place when the number of deployed Data Mart attributes exceeds the number of included deployed Data Mart attributes.

In addition, each package includes a fixed number of Yotilla instances. Metering of the additional Yotilla instances only takes place when the number of started Yotilla instances exceeds the number of included Yotilla instances.
